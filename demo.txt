
1. git init 			-> used to intialize when to create new repository locally
2. git status 			-> used to check the branch and working status of file 
3. git add <file_name>		-> add file_name to the staging area 
4. git add . 			-> add all updated files to the staging area
5. git ls-files			-> to check files available	
6. git commit -m "first commit"	-> commit the changes made with the message





7. git branch 			-> to see available branches in git 
8. git branch Branch_1		-> created new branch named as Branch_1
